# ALICE-Optics

ALICEエコシステム向けの光学シミュレーションライブラリ。幾何光学、波動光学、偏光、ファイバー光学、収差解析を純Rustで実装。

## 機能

- **幾何光学** -- スネルの法則(スカラー/ベクトル)、フレネル方程式、ブリュースター角/臨界角
- **レンズ系** -- 薄肉/厚肉レンズ方程式、レンズメーカーの式、ABCD光線伝達行列、近軸光線追跡
- **回折** -- 単スリット/二重スリット強度、エアリーディスク、レイリー分解能、回折格子
- **偏光** -- ジョーンズベクトル/行列、ストークスベクトル、ミューラー行列、波長板、偏光子
- **薄膜干渉** -- 反射率、反射防止膜設計、強め合い/弱め合いの波長
- **ファイバー光学** -- 開口数、V値、モード数、減衰、モード分散
- **収差** -- 球面収差、色収差、コマ、ペッツバール曲率、歪曲、アッベ数、色消しダブレット
- **ガウシアンビーム** -- レイリー範囲、ビームウエスト、発散角、グイ位相
- **プリズム・干渉計** -- 最小偏角、角度分散、光路長差、コヒーレンス長、フリンジ可視度

## アーキテクチャ

```
Ray / Vec3           幾何コア (スネル, フレネル, レンズ)
    |
Matrix2x2            ABCD光線伝達行列
    |
JonesVector/Matrix   偏光 (ジョーンズ計算)
StokesVector         偏光 (ミューラー計算)
    |
Diffraction          波動光学 (スリット, 格子, エアリー)
    |
Fiber / Aberration   応用光学モジュール
```
